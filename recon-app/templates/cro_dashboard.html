Formula & Source

Click any metric on the left to see how it's calculated.

Each metric in this dashboard is derived from the live PostgreSQL replica that the MT5-CRO-Backend keeps in sync with the MT5 Manager API.

The three status sections share the same underlying formulas — only the time window and data source change:

  • Today's Status reads live accounts_snapshot, positions_snapshot, and exposure_snapshot tables (refreshed every slow cycle, ~60s).
  • Yesterday's Status uses the prior UTC day's daily_reports rows, plus deposits/withdrawals and bonuses cumulative through yesterday end-of-day.
  • Month-to-Date aggregates closed trades and floating P&L changes from the start of the current UTC month.
